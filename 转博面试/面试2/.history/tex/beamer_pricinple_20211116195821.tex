\title{幻灯片原则}
写幻灯片报告需要遵守的原则.
\begin{enumerate}
  \item 首页把相关人物都写清楚.
  \item 短时间的幻灯片结构保持简单,重要信息放在幻灯片标题.
  \item 报告中目录换提纲,目录不使用小众名词.
  \item 减少幻灯片的字数,避免换行.
  \item 区分幻灯片内容和报告口语内容.
  \item 已发表论文从期刊中截图.
  \item 项目叙述顺序:背景,意义,是什么,做了什么.
  \item 研究背景先提出研究解决的问题和问题的重要性.
  \item 数学报告中不包含编程内容.
  \item 报告中不加凑数,降低评价的成果和经历.
  \item 使用缩写前需要定义.
  \item 一张幻灯片内容不能太少太空,缩减或者用图片填充.
                                                                                                                                                                                                                          
\end{enumerate}
